Este proyecto fue hecho para un curso de ComIt, es una calculadora cientifica con un graficador
La calculadora la hice yo de principio a fin al igual que el diseño, el graficador tuve mucha ayuda
Es necesario empezar por el login, no por el index, porque sin una cuenta no se pueden usar ciertas partes de la página, hecho asi a proposito
Tiene un sistema de login conectado a una base de datos escrita en MySQL, al igual que un historial y un perfil de usuario basico.
